Vous avez besoin de [Node.JS](https://nodejs.org) (testé avec la version v24.11.1).

Pour installer les dépendances, fait la commande ci-dessous dans le répertoire du projet :

```bash
npm i
```

Pour lancer le serveur, faite la commande ci-dessous dans le répertoire du projet puis aller sur ce [lien](http://localhost:8080) :

```bash
node server
```
